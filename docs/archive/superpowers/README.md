# Historical Superpowers Plans

The files under `plans/` are historical design and implementation records.
They are retained for audit context, but are not executable runbooks.

The `superpowers` source is intentionally disabled in the current skill
registry. Do not follow old `REQUIRED SUB-SKILL` instructions in these files.
Use the current public runbooks under `docs/`, especially
[`skill-supply-chain.md`](../../skill-supply-chain.md), and the managed Matt
Pocock workflow when applicable.
