# Issue tracker: GitHub

Issues and specs for this repository live in GitHub Issues:

- Repository: `suainam/mac-bootstrap-template`
- CLI: `gh issue ...`
- Spec and ticket publication: create one GitHub issue per artifact.
- Tickets use the `ready-for-agent` label.
- Blocking edges are recorded in each ticket's `Blocked by` section. Use native
  GitHub issue dependencies when available; otherwise keep the textual edges.
- PRs are not a triage request surface.

The repository's public template is the implementation boundary. Do not put
private parent-repository paths, credentials, subscriptions, or machine facts
in issue bodies or committed documentation.
