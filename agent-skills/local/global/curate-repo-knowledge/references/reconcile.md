# Reconcile

Use this branch when a handbook exists or knowledge must be synchronized after change.

## Reconciliation pass

1. Measure persistent and routing files before editing.
2. Enumerate active knowledge surfaces and their declared owners. Exclude clearly archived history from current-fact checks.
3. Compare current code, catalogs, schemas, configuration, tests, and accepted decisions with the claims they own.
4. Classify findings as duplicate, stale, conflicting, misplaced, dead reference, missing affected audience, narrative sediment, or budget overflow.
5. Trace the current diff or completed milestone through its affected audiences. Update the authority first, then its links or projections.

## Editorial order

Apply this order: remove stale material; merge duplicates into their authority; promote stable knowledge from memory or temporary plans; replace copied facts with links; add genuinely missing knowledge last.

A fact graduates when it repeatedly recurs, explains how the system works, or must be known by the next maintainer. Promote it to the correct project authority, then remove or reduce the transient copy to a pointer. Keep event history in version control, issues, changelogs, retrospectives, or archived plans.

Keep Agent rules limited to constraints whose absence changes Agent behavior: red lines, non-obvious boundaries, critical commands, permission rules, routing, and recurring traps. Move mechanisms, tutorials, event narratives, and tool-enforced lint rules to their proper surfaces or remove them.

## Mutation classes

- **safe**: reversible link repair, verified compatibility symlink creation, or bounded correction with one unambiguous authority.
- **review-required**: semantic rewrite, ownership move, inferred rule, conflict resolution, or budget-driven split.
- **prohibited until explicitly approved**: authority overwrite, deletion, rename, hooks, global Agent configuration, destructive cleanup, or weakened guardrail.

Reconcile is complete when each active fact has one owner, affected audiences route to it, persistent budgets pass, and every unresolved semantic finding appears in the summary.
