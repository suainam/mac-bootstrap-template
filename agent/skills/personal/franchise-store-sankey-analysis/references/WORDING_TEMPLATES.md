# Wording Templates

Use these as templates to mirror the established business style. Replace only the numbers and sign.

## Report template

```md
1. 背景：{项目背景/上线范围}，公司对加盟店的发货毛利 {发货毛利变化}（{发货同比}）（{口径说明}）
2. 组货优化了门店商品结构，目录内商品销售额占比 {销售占比变化}（{销售占比旧值}->{销售占比新值}），毛利额占比 {毛利占比变化}（{毛利占比旧值}->{毛利占比新值}），商品数占比 {商品数占比变化}（{商品数占比旧值}->{商品数占比新值}）
3. 门店总盘层面，销售{增长或减少}{销售变化额}（{销售旧值}->{销售新值}，{销售同比描述}）、毛利{增长或减少}{毛利变化额}（{毛利旧值}->{毛利新值}，{毛利同比描述}）、近90天周转{改善描述}（{周转旧值}->{周转新值}）。整体看，{一句总括}
4. 归因分析：
①26年较25年整体销售{增长或减少}{净变化额}，主要来源是：26年目录内带来{正向贡献描述}，减去{存量盘描述}，减去{目录内汰换描述}，减去{目录外汰换描述}，最终剩余{净结果描述}
公司发货毛利：{发货毛利结论}
```

## Sankey explanation template

Use `总 -> 分 -> 总`.

```md
这张桑基图完整展示了 2025 到 2026 年商品在目录内、目录外的新增汰换状态之间的流转。整体看，{一句总括}。

1. 首先看存量部分，目录内和目录外的商品销售从 25年的 {存量旧值} 下降到 26年的 {存量新值}，{变化描述}，说明未发生目录调整的基础盘本身是在{下滑/改善}。

2. 在这个背景下，26年组货项目通过两类动作实现了结构改善：
1）将25年原目录外表现较好的商品引入26年目录内，这部分销售从 {外转内旧值} 提升到 {外转内新值}，表观{增长或减少} {外转内变化值}；
2）26年新增目录内商品，新增 {新增条目数} 个品，贡献销售约 {新增目录内销售}。
3）26年的目录内汰换 {目录内汰换值}（原25年目录内）
4）26年目录外 {目录外承接值}，其中 {拆分说明}。

结论：26年目录内带来{正向贡献描述}，减去{存量减少描述}，减去目录内汰换的 {目录内汰换值}，减去目录外汰换的 {目录外汰换值}，最终剩余 {净结果描述}
```

## Wording rules

- Prefer `汰换`; avoid `损耗`
- Prefer `%`; avoid `pct`
- If total is negative, prefer `负增长` / `承压`
- Do not overplay `目录内周转` or `目录内动销率` when `目录内商品数` sharply expands, unless the user asks
