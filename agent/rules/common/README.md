# Common Rules (Language-Agnostic)

These rules apply to all languages and projects.
Symlinked to `~/.claude/rules/common/` — Claude Code loads these as always-on rules.

## Contents

- `coding-style.md` — Immutability, file organization, naming
- `git-workflow.md` — Commit format, PR process
- `testing.md` — TDD, coverage requirements
- `hooks.md` — Hook architecture, lifecycle events
- `security.md` — Security-first development patterns

## Rule Application

Rules in this directory are loaded automatically by Claude Code.
They supplement the 12 core operating rules in `~/.claude/CLAUDE.md`.
